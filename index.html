ICAO adopted ACR-PCR in 2020 as a replacement for ACN-PCN, with full applicability scheduled for 28 Nov 2024. It uses layered elastic analysis rather than simplified CBR/Westergaard methods. Operational Notes // AFTTP 3-3.C-17 Table 3.1

  • Data from 1C-17A-1-1, Ch 10 "Support Area Operations"
  • No CG limitations applied. FWD CG <32% requires TO check
  • Normal tire PSI: 144 (main), 160 (nose)
  • Values >447K: check 1C-17A-1-1 Sec 5 for SPRO limits
  • Note 1: Overload <5% annual. Asphalt +10%, Concrete +5% PCN
  • Note 2: Use L=40/Thickness=20 if unknown. Overload +25%
  • Note 3: Prior permission required when ACN exceeds PCN
  • ACN interpolated linearly between OEW and MRW (ICAO method)
  • COMFAA anomaly: rigid ACN can invert on high subgrades. Use TO values.
