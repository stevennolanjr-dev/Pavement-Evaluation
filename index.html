ICAO adopted ACR-PCR in 2020, replacing ACN-PCN, with full applicability Nov 2024. Uses layered elastic analysis vs. older CBR/Westergaard methods. No mathematical conversion exists between PCN and PCR values. AFTTP 3-3.C-17 (Jul 2025) still uses ACN-PCN. This tool remains valid for USAF mission planning. If an airfield publishes only PCR, contact the host nation for an equivalent PCN estimate or compare ACR at planned weight to PCR directly using TO 1C-17A-1-1 Figure 10-2.

Operational Notes // TO 1C-17A-1-1 Ch 10 / AFTTP 3-3.C-17

  • PCN/LCN/CBR data from AFTTP 3-3.C-17 Table 3.1. CBR CG data from TO Fig 10-5.
  • No CG limitations applied to PCN/LCN. FWD CG <32% MAC requires TO check.
  • Normal tire PSI: 144 psi main (138 unloaded), 160 psi nose (155 unloaded).
  • Values >447K lbs: check 1C-17A-1-1 Sec 5 for SPRO limits.
  • Note 1: Overload <5% annual. Asphalt +10%, Concrete +5% PCN.
  • Note 2: Use L=40 / Thickness=20 if unknown. LCN overload +25% (~1 flt/day).
  • Note 3: Prior permission required when ACN exceeds published PCN.
  • ACN interpolated linearly between OEW (282K) and MRW (586K) per ICAO method.
  • COMFAA anomaly: rigid ACN can invert on high subgrades. Use TO values.
  • OVLD percentages modeled from TO 1C-17A-1-1 Figure 10-8. Use for planning only.
